# NeuroScan AI Documentation

## Overview
NeuroScan AI is an advanced Alzheimer's detection system using dual AI models to analyze brain MRI scans.

## Quick Start

### Prerequisites
- Python 3.8+
- MySQL 8.0+
- 4GB RAM

### Installation
```bash
git clone https://github.com/Godwinisravel/NeuroScanAI.git
cd NeuroScanAI
pip install -r requirements.txt
python app.py
