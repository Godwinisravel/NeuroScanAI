# 🧠 NeuroScan AI - Advanced Alzheimer's Detection System

A professional web-based AI system for Alzheimer's disease detection using dual-model analysis of brain MRI scans.

## ✨ Features

- **Dual AI Model Analysis** - Compares specialized CNN vs. EfficientNet models
- **Interactive Dashboard** - Role-based portals for Patients, Doctors, and Admin
- **Professional PDF Reports** - Medical-grade ultrasound-style reports
- **Dark/Light Theme** - Comfortable viewing in any environment
- **Complete User Management** - Registration, authentication, and roles
- **Advanced Visualization** - 4 comparison graphs including pie charts and bar charts
- **Medical Recommendations** - Stage-based clinical recommendations
- **Report Management** - Delete individual or bulk reports
- **Responsive Design** - Works on desktop, tablet, and mobile
